Salvar e Recuperar dados

*Os dados são salvos como STRINGS
*Utilizar a function Number() para converter para número

Salvar - setItem()

Este Método contém dois parâmetros:

  • Chave: nome da variável
  • Valor: conteúdo da variável

Exemplo:

localStorage("idade", 17)

Recuperar - getItem()

Este Método contém um parâmetro:

  • Chave: nome da variável

E uma variável para receber o valor

Exemplo:

let idade = localStorage.getItem("idade")

Convertendo para número:

let idade = Number(localStorage.getItem("idade"))
