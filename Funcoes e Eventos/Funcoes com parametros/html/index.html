• Funcões com Passagem de Parâmetro

  • Uso do arguments

    Funciona como um vetor, contendo a lista dos valores passados na chamada da função. Dessa forma, pode-se omitir a passagem dos parametros

  • Funções com Retorno de Valor

    Utiliza-se o comando return seguido do conteúdo do retorno.

    É possível utilizar uma variável para receber o retorno de uma função.
